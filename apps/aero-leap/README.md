# Aero Leap 🪷

Frutiger Aero 風のグラス・リープゲーム。プレイヤーはガラスのフィギュアで、足場を
チャージ＆ジャンプで渡っていく。コースを進むほど世界が次々と切り替わる（1本道で、
「何もない道」は無し）。素の **HTML / CSS / JS + Three.js**（import map / unpkg）なので
Netlify にそのまま置くだけで動く（ビルド不要・TypeScript 不要）。

## コース（連続した1つのワールドを Z 方向に進む）
進むと空・水・光・霧が次のコースのパレットへ滑らかにブレンドする。前方には
フォグ越しに次のコースが見えてくる。

1. **Aero Pond**（現状保護）— 睡蓮の葉・東屋・噴水・巻きひげ植物
2. **Helvetica Aqua** — 水中。熱帯魚の群れ・海藻・太陽光線・虹。**サンゴ／気泡膜が足場**。
   キャンディ色（キーライム/ライム/ブルーベリー/タンジェリン/インディゴ）の
   **ガラスのイルカ**が跳ねる。途中に**イルカに乗って海を渡る**区間あり。
3. **Technozen** — 緑の Zen。大きな葉・てんとう虫・ガラス球・浮遊キューブ。**葉／緑のモチ(バネ)が足場**。
4. **Cybercore** — 氷河＋白い未来都市・ガラス塔・鏡面メタルフロア・**ガラス階段(エスカレータ)**。
   **光るガラスキューブが足場**。足場が上下に起伏して立体的。
5. **Jungle Aero** — **滝**・岩・ヤシ・霧・太陽光線。**岩／蓮の葉が足場**。
6. **Retro Apple** — 半透明プラスチック世界。**iMac G4 のドーム**と**iPod の天面**に乗ってジャンプ。
   **クリックホイールが回転**（乗ると速く回る）。
7. **Clear**（ゴール）— ガラスのエスカレーター・**ガラスの貝**・**Aero テクスチャ**のパネル・
   Aero 柄の回るクリックホイール iPod・飛び回るガラスのイルカ。**最後の足場でクリア → OSX音**。

## 操作 / ギミック
- **押しっぱなしでチャージ → 離してジャンプ**（飛距離＝チャージ量、近い足場に自動アシスト）。
  PC はクリック（押して離す）でも同じ。
- **画面の左/右側を押す**と、その側にある足場・オブジェへ寄せて飛べる（おまけアシスト）
- **空中でタップ＝レスキュー二段ジャンプ**（1ジャンプにつき1回）
- **モチ（バネ）**に乗ると次の足場へポンと弾む。色＝水色 / 濃い青 / 緑（Technozen の緑足場）
- **滝の噴き上げ（updraft, Jungle）**に触れると一気に上へ打ち上がる
- **ハートの回復アイテム**に触れるとどっと弾けて **ハート+1（最大5）**
- **リングをくぐる**とボーナス
- **左のコースレール**のドットをタップで、到達済みコースへ自由に行き来できる（前にも先にも）
  （`N` キーで Retro Apple に一気にワープする隠しショートカットあり）

## 触覚（iOS）
画面どこでも／各ボタンの裏に、透明な**本物の `<input type="checkbox" switch>`** を重ねてある。
指が当たるのは本物のスイッチなので「ユーザー自身がトグルした」扱いになり、
iOS 26.5+ / 旧 iOS どちらでも **1タップ＝確実に1発の触覚**が鳴る。

## 音
**http(s)（Netlify）では Web Audio でギャップレス・ループ。`file://` では `fetch` が
ブロックされるので `<audio>` に自動フォールバック**（だからローカルでもちゃんと鳴る）。

BGM：
- **最初＝`iloop1`（loop1）**。Cybercore / Retro Apple は `iloop2`（loop2）。
- **イルカに乗っている間は `iloop2`** → 降りたらコースの曲へ。

SFX：
- ジャンプ＝`NotificationAudioTone.wav` / 着地＝`MRUIFeedbackTypeSliderSlidToMinimumValue.wav`
- 左の足場に着地＝`Note_03_Right_F4_…wav` / 右の足場＝`Note_03_Left_G5_…wav`
- イルカ乗車＝`Calibration_tone_success.wav` / クリア完了＝`MU_Out18a.m4a`

自動再生は最初の「Hello」タップで解禁。右下の ♪ でミュート。

## Hello イントロ
`Desktop/HelloText/en.json` のストロークを使った **手書きアニメの "Hello"**（ガラス白＋グロー）を
**透明 blur のみ**の上（背後の3Dシーンがぼけて見える）に描画。タップ/クリック/キーでゲーム開始。

## 収集 & ビルダー（G4 Cube → Dock → コース構築）
- コースに **G4 Cube**（透明アクリルの立方体に中身を閉じ込めたアイテム）が出現。触れると **OS X 10.4 風 Dock** に貯まる。
- **クリア時に Dock が登場**。集めた中身：🪷睡蓮 🪨岩 💧水滴 🟦ガラスキューブ 📱iPod 🌉橋 🐬イルカ 🖥️iMac 🏢ビル 🍎Apple Store 🪙コイン ❤️ハート。
- 左上の **🫨**：タップで **ビルドモード**切替（作る↔遊ぶ）／**長押しで `tog.html`** を開く。
- ビルドモード：Dock からアイテムを選び、画面をタップで地面に配置（プラットフォームは乗れる）。**🧽 消しゴム**で削除、下の **スライドで全リセット**。
- 置いた島はリスタートしても残り、インベントリも維持（集めて作って遊ぶを1つに）。
- 動く橋・ガラスビル・Apple Store・コインなど Frutiger Aero 的テクスチャを同梱。

> 注：3D が重め（透過ガラス多数）。実機GPUなら可だが、モバイルで重い場合は decor 数や `transmission` を調整。
